# manabi-app

個人的に作った学習アプリを、手持ちの端末で使えるように GitHub Pages で配布しているものです。

- ブラウザだけで動きます。インストールもアカウント登録も不要です
- 各アプリは 1ファイル完結の HTML で、**学習記録は使っている端末のなかだけ**に保存されます
  （どこかのサーバーに送られることはありません）

入口: https://yokahana.github.io/manabi-app/

## アプリ一覧

### 🌲 森とグミ — TOEIC 750 → 850

[toeic.html](https://yokahana.github.io/manabi-app/toeic.html)
／ [取扱説明書](https://yokahana.github.io/manabi-app/toeic-guide.html)

社会人向け。TOEIC 750点前後から850点を目指すための語彙・文法・読解の練習。
使い方は取扱説明書のほうが詳しいです。

### 学習アプリ（数学・英語）

| アプリ | 対象範囲 |
|---|---|
| [🩰 ダンスステージ](https://yokahana.github.io/manabi-app/chu1.html) | 中学1年の数学・英語 |
| [⚽ ビクトリーロード](https://yokahana.github.io/manabi-app/chu3.html) | 中学3年の数学・英語（受験範囲まで） |
| [🏆 バトルアリーナ](https://yokahana.github.io/manabi-app/sho4.html) | 小学4年の算数・英語（英語は聞き取り中心） |

出題は10問1セット。正解でXPが貯まりレベルが上がります。
間違えた問題は忘れかけたころに出しなおす仕組み（間隔反復）で復習されます。

## 使い方

1. 上のリンクを開く
2. ブラウザの共有メニューから **「ホーム画面に追加」** する
   （iPad / iPhone は Safari、Android は Chrome。PCならブックマークで十分です）
3. あとはホーム画面のアイコンから起動する

**記録は開いたブラウザごとに保存されます。** 同じ端末・同じブラウザで使い続けてください。
端末やブラウザを変えると、それまでの記録は引き継がれません。

機種変更などで引き継ぎたいときは、各アプリの設定画面（学習アプリは保護者画面）から
記録を**書き出し**て、移行先で**読み込み**ます。同じ画面から記録の消去もできます。
